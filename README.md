
# Express-Boilerplate

A crawler for https://minha.unesc.net/

## Quickstart

```
$ git clone this repository
$ npm install
$ UNESC=[login]:[password] npm start
example: $ UNESC=12345:abc123 npm start
```
Then go visit http://localhost:8000
